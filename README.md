<summary><b>📌 JobTracker — Application Pipeline Management System</b></summary>

<br/>

A full-stack web application that helps job seekers organise, track, and analyse their entire job search pipeline in one place. Built out of a real need during my own placement season — because spreadsheets break down fast when you're applying to 50+ companies.

| Attribute | Details |
|:---|:---|
| **Stack** | Java · Spring Boot · MySQL · React · REST API |
| **Scale** | Multi-stage pipeline tracking (Applied → Interview → Offer → Rejected) |
| **Performance** | Fast CRUD operations with JPA/Hibernate, pagination support |
| **Security** | Session-based auth, input validation, role-scoped data access |
| **Impact** | Personal productivity tool actively used during placement cycle |
| **Repository** | [github.com/sharv2884/jobtracker](https://github.com/sharv2884/jobtracker) |

The application models each job application as a stateful entity that moves through a defined pipeline. The backend exposes a clean REST API consumed by a React frontend. Data persistence is handled by Spring Data JPA with MySQL. The project gave me deep practice in building real-world CRUD applications with business logic beyond the tutorial level.
